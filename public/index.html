This is the official Nathan Cobb website, used to display my previous work, my personal and/or school projects, and whatever else I decide to put on here. Info About the Site

This site is (currently) a static html page being served by ExpressJS. Eventually I want to switch to a rust-based web server like Axum or Actix, but I wanted to get something started up quick. The web server is hosted on a Debian VM mangaged by Proxmox. All of this is run on a Cisco UCS C220 M3 which was donated to me by a coworker. I use nginx as a reverse proxy to access the web server and handle any other calls needed. Along with eventually switching from expressjs to a rust web server, I do also want to switch from static html to react for the frontend.
