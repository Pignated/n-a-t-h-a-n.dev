Info About the Site

This site is (currently) a static html page being served by ExpressJS. Eventually I want to switch to a rust-based web server like Axum or Actix, but I wanted to get something started up quick. The web server is hosted on a Debian Container mangaged by Proxmox. All of this is run on a Dell PowerEdge R720. I use nginx as a reverse proxy to access the web server and handle traffic for other systems of mine. Along with eventually switching from expressjs to a rust web server, I do also want to switch from static html to react or something similar for the frontend.
